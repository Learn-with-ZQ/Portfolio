# Tech Stack Summary

## Languages
- TypeScript
- TSX for React components
- CSS/Tailwind for styling

## Frameworks and libraries
- Next.js 15 (App Router)
- React 19
- Tailwind CSS 4
- Framer Motion
- `next-themes`
- `lucide-react`
- `clsx`
- `tailwind-merge`

## Package manager
- npm

## Bundler / runtime
- Next.js build pipeline
- No separate webpack/Vite configuration is needed in this repo

## Linting / formatting
- ESLint via `eslint-config-next`
- Prettier with Tailwind plugin

## Deployment platform
- Vercel is the assumed production host
- Site URL is sourced from `NEXT_PUBLIC_SITE_URL` or the default Vercel URL

## Analytics
- No analytics scripts or tracking libraries are present
